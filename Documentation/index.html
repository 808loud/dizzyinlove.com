Introduction

John and Jane is Responsive Bootstrap 3 Wedding Event Template built with Bootstrap 3.3.2, HTML5 and CSS3. Well Organized and valid Code.

JavaScript and Ajax

John and Jane imports following Javascript files.

  2. Custom scripts
  3. Google Map Api
  4. Masonry
  5. ImagesLoaded
  6. Magnific POPUP

All code are beautifully written. You can customize easily. Just look the js code.
Example: If you want to Customize Countdown, just find the following block into the script.js and change what you need:
 

//----------------------------------------------------

//--------------------Countdown-----------------------

//----------------------------------------------------


$("#count-down").TimeCircles(

   {   

     circle_bg_color: "#fd6e6e",

     use_background: true,

     bg_width: 1,

     fg_width: 0.03,

     time: {

          Days: { color: "#fefeee" },

          Hours: { color: "#fefeee" },

          Minutes: { color: "#fefeee" },

          Seconds: { color: "#fefeee" }

      }

   }

// receiver email address
